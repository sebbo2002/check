### check_kue

```
Usage: check_kue [Options]
Options:
  -h, --host=<STRING>         kue-URL (`https://kue.example.com`)
      --disable-http-warning  Disable warning for kue instances without SSL
      --help                  display this help
```

#### Example

```
check_kue -h "https://kue.example.com"
```
