### check_teamcity

```
Usage: check_teamcity [Options]
Options:
  -h, --host=<STRING>         TeamCity-URL (`https://teamcity.example.com`)
      --disable-http-warning  Disable warning for TeamCity instances without SSL
      --help                  display this help
```

#### Example

```
check_teamcity -h "https://tramcity.example.com"
```
